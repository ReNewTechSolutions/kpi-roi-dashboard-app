## 9) What a Buyer Needs to Do (reduces support burden)
1. Create Supabase project
2. Run `schema.sql` then `rls.sql`
3. Set Vercel env vars:
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
4. Deploy
